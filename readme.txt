Edit title tags and meta descriptions for posts, pages, and custom post types.

== Description ==

A simple and lightweight plugin that enables you to edit title tags and meta descriptions on posts, pages, and custom post types. Includes a character counter on input fields.

= Automatic Installation =

1. Login to your Dashboard as admin.
2. Once logged in select Plugins -> Add New.
3. Upload Conversions Meta Tags and activate.
4. Edit meta tags on each page/post using the metabox at the bottom of the post editor.

= Manual Install =
1. Extract the contents of conversions-meta-tags.zip and upload to the wp-content/plugins folder.
2. Login to your Dashboard as admin.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Edit meta tags on each page/post using the metabox at the bottom of the post editor.
